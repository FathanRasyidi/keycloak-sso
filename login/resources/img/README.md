# Logo Resources

This directory contains logo images used in the Keycloak login theme.

## Required Files

- `logo-pemda-diy.png` - Logo Pemerintah Daerah DIY (Daerah Istimewa Yogyakarta)

## Current Status

A placeholder SVG logo (`logo-pemda-diy.svg`) has been provided for development purposes.

**IMPORTANT**: Replace with the actual logo file:
1. Add the official `logo-pemda-diy.png` file to this directory
2. Ensure proper permissions and file format

## Fallback Behavior

If the PNG logo is not found, the template falls back to:
```
https://jogjaprov.go.id/storage/files/shares/page/1518066730_2d84b769e3cc9d6f06f8c91a6c3e285c.jpg
```

## Recommended Image Specifications

- Format: PNG with transparent background
- Dimensions: 200x200 pixels (or maintain aspect ratio)
- File size: < 100KB for optimal loading
- Color mode: RGB or RGBA (for transparency)
